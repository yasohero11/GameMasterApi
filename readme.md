# GamesMaster API

> Backend API for GamesMaster application, which is a Games directory website

## Usage

Rename "config/config.example.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
